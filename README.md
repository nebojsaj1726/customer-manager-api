## Description

REST API for Customer Manager, built with NestJS and Mongoose

## Installation

```bash
$ npm install
```

## Docker

There is a docker-compose.yml file for starting MongoDB and Mongo Express

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
